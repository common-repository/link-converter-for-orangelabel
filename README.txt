== Description ==

= What is the  Link Converter for Orangelabel? =

The link converter changes any (not disabled) outgoing links to Orangelabel links, automatically adding the affiliate tracking parameters where possible.

== Installation ==

1. Upload the `Orangelabel-Link-Converter` directory to the `/wp-content/plugins/` directory
   or simply download it through your plugin browser
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your orangelabel id for each network you received from your account manager through Awin
